## 🎨 CONSISTENT ART STYLE

### Visual Identity
- **Style**: Stylized cartoon with soft glow effects
- **Inspiration**: Mobile AAA games (Clash Royale, Brawl Stars aesthetic)
- **Rendering**: Clean edges, vibrant colors, soft bloom lighting
- **Format**: UI-ready, centered compositions, 16:9 aspect ratio
- **Depth**: 3-layer parallax (foreground/midground/background)

## ⚔️ MODE 1: SOLO - "THE LONE KNIGHT"

### Character Design
```
NAME: The Lone Knight ("Focus")
ROLE: Solo Mode Mascot

APPEARANCE:
- Slim, elite assassin-knight build (NOT bulky)
- Purple glowing armor with sleek plating
- Glowing purple sword with soft pulse animation
- Hooded helmet with glowing visor slit
- Cloak flowing in wind
- Confident, calm stance

COLORS:
- Primary: #6c63ff (Purple)
- Secondary: #a855f7 (Magenta)
- Glow: rgba(108, 99, 255, 0.6)
- Armor: Dark purple with light purple edges

PERSONALITY:
- Calm, confident, focused
- Silent but powerful presence
- Precision-oriented

ANIMATIONS:
- Idle: Sword glows slowly (breathing effect)
- Selected: Raises sword, purple energy burst
- Attack: Quick precise strike with purple trail
- Victory: Sword pointed skyward, lightning cracks
```

### Background Environment
```
THEME: Floating Islands in Purple Storm

LAYER 1 - FOREGROUND:
- Faint glowing particles drifting slowly
- Purple crystal fragments floating
- Soft mist at bottom edge

LAYER 2 - MIDGROUND:
- Main floating island platform
- Ancient stone architecture
- Knight statue silhouette

LAYER 3 - BACKGROUND:
- Multiple floating islands in distance
- Purple storm clouds
- Slow-moving lightning bolts
- Purple gradient sky

EFFECTS:
- Lightning cracks every 3-4 seconds
- Floating particles always moving
- Parallax scroll on mouse movement
```

## ✨ VISUAL EFFECTS SPECIFICATIONS

### Particle Systems Per Mode:

**Solo**: Purple floating crystals, slow lightning
**Survival**: Red embers, falling ash, smoke
**Blitz**: Gold sparks, speed lines, electricity
**Rush**: Cyan-green light trails, forward arrows
**Conquest**: Holographic hexagons, connection lines
**Mixed**: Glitch particles, color-shifting orbs

- Idle: 2-3 second loops
- Selected: 0.5s burst
- Attack: 0.3s fast action
